So you're having problems running SuperGT. Well, see if your problem is listed here. If it isn't, drop me a
note at my e-mail address.


Minimum System Requirements:

500 MHz Pentium III or equivalent
64 MB Ram
2 MB HD (30 if you had to install DirectX)
Video card with at least 8 MB of memory


Recommended System Requirements:

700 MHz Pentium III or equivalent
64 MB Ram
2 MB HD (30 if you had to install DirectX)
Video card with at least 32 MB of memory
Windows-compatible Gamepad


When I start the game, it says
AUTOMATION ERROR.

The most likely thing is your video card does not support 640x480 at 16 bit color. If it doesn't, you must have a really old computer.


The game runs really s l o w .

You probably don't have a great 3D accelerator card, or you need to change some of the settings for the game. Run Config.exe and select a low fullscreen resolution, like 320x240


I get Object Variable or With block not set

You need to get DirectX, see below


The sound is really choppy.

You probably don't have the latest DirectX drivers for your sound card. Go to http://www.microsoft.com/directx


How'd you make the game?

I used Visual Basic 6.0 and one speedy 900-MHz PC with 192MB ram, a 32MB TNT2 graphics card, and Windows XP. Nothing spectactular, but the Duron CPU is fast enough for everything I throw at it.


Did you make the car model?

No. I used 3000gt.3ds off of 3DCAFE's site.
